Ni WooCommerce Order Export plugin provides the functionality to export the sales order information into CSV or excel format.
 
== Description ==

<h3>Ni WooCommerce Order Export Description</h3>

<p>
<strong>Sales order report and export</strong>
Sales order report shows all sales information in tabular or grid format like billing details, shipping details, order total and sales order  product.

Order billing report display all billing details in tabular or grid format like billing name, email, contact number  country and many more fields

== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
2. Activate "Ni WooCommerce Order Export" plugin from the admin panel.
3. Find "Order Export" in the admin menu.
